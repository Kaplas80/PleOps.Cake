    v0.4.1 (March 13, 2021)

    As part of this release we had 3 issues closed. It introduces compatibility with Ubuntu 20.04 (LTS) which is the new image in Azure and GitHub build pipelines.

    Migration steps:

    • Upgrade Cake to version 1.1.0 by running: dotnet tool update cake.tool.
    • Update the dockerfile for the dev container (.devcontainer/Dockerfile) to use the official .NET 5 image.
    • Update your GitHub workflow (.github/workflows/build-and-release.yml) to use the latest .NET SDK.

    Enhancements

    • #43 Upgrade Cake and dependencies to support Ubuntu 20.04
    • #42 Support test setting file from other paths or missing
    • #35 Support Ubuntu 20.04 Focal
